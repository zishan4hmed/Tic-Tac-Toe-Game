# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## 📖 Description

This project is a browser-based Tic Tac Toe game for two players. Players take turns marking X and O on a 3×3 grid. The game automatically detects a winner or a draw and provides an option to restart the game.

## ✨ Features

- Two-player gameplay
- Winner detection
- Draw detection
- Restart / Reset button
- Responsive design
- Simple and clean user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
